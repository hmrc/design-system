    Anti-dumping duty measures

    Document type

    Publication: guidance (with a HTML attachment)

    When to use

    Use this pattern to tell users about a change in import or export procedures.

    How it works

    You do not need to mention 'import' or 'export' unless really necessary, as anti-dumping duty only apply to imports into the UK and EU.

    Publication

    Title

    Description (only include the object or item and where it’s imported from or describe what the measure is about and the measure number in brackets at the end).

    Check the anti-dumping duty measures collection to make sure your title has not already been used in a recent measure.

    Examples of good titles:

    • Hand pallet trucks imported from China (anti-dumping duty xxxx)
    • Name change of a company that exports ceramic tableware and kitchenware from China (anti-dumping duty xxxx)

    Summary

    Provide a short description of what the attached document is about.

    Do not start your summary with 'Find out'.

    For example, ‘Changes to the duty codes for trucks with a weight indicative system, originating in the People’s Republic of China’.

    Details

    Enter this text:

    Copy 
    Anti-dumping duty is an import duty charged in addition to normal Customs Duty and is applied in the UK and across the whole EU. It’s designed to allow the UK and EU to take action against goods that are sold at less than their normal value – that being defined as the price for ‘like goods’ sold in the exporter’s home market.

    HTML attachment

    Although measures contain technical information you should be able to get a general understanding about what they are for - go back to the subject matter expert if you think a measure does not make sense or is unclear.

    Check any hyperlinks in the business draft. Contact the subject matter expert to get correct details if any links:

    • point to the home page of the EurLex website, rather than the page containing the relevant legislation
    • go directly to a PDF (or other file), rather than the webpage that houses it

    Title

    Use the same title as the publication.

    Body text

    Enter the contents of the anti-dumping duty measure.

    Tagging and collection page

    Make sure you:

    • add the appropriate associations and specialist sector tags
    • add the new measure to the anti-dumping duty measures document collection - do not create a public change note

    If you need to change this pattern

    You should send a request to the Editorial Board if you need to change this pattern.
