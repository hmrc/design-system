    Anti-Dumping Duty measures

    Document type

    Publication guidance - stub page and HTML attachment.

    How it works

    Stub page

    Title

    Describe the measure and include the reference type and number in brackets.

    For example, ‘Hand pallet trucks imported from China (Anti-Dumping Duty 2221)’.

    Summary

    Provide a short description of what the attached document is about.

    For example, ‘Changes to the duty codes for trucks with a weight indicative system, originating in the People’s Republic of China’.

    Details

    Enter this text:

      Anti-Dumping Duty is an Import Duty charged in addition to normal Customs Duty and is applied across
  the whole EU. It is designed to allow the EU to take action against goods that are sold at less than their
  normal value - that being defined as the price for ‘like goods’ sold in the exporter’s home market.

    Collection

    Add the page to the Anti-Dumping Duty measures collection page.

    HTML attachment

    Title

    Describe the measure and include the reference type and number in brackets.

    For example, ‘Hand pallet trucks imported from China (Anti-Dumping Duty 2221)’.

    Body text

    Enter the contents of the Anti-Dumping Duty measure.
