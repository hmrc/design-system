    Government Gateway user ID and password

    When to use

    Use this pattern when you're linking to a service that requires the user to sign in using Government Gateway.

    Where to place

    You should place this pattern before the link to the service, unless there's a design pattern that says different.

    How it works

    Option 1

    Use this option for most services.

    Open this example in a new window
    • Markdown
    Copy 
                      To XXX, you need a Government Gateway user ID and password. If you do not have a user ID, you can create one when you XXX.

    Option 2

    Use this option if the user needs to use specific credentials. For example, if they must use the user ID and password they created when they registered for a tax, or enrolled for online service.

    Open this example in a new window

                      You’ll need the Government Gateway user ID and password you used when you XXX.

    Option 3

    Use this option in links to services from publication pages which use an external link attachment.
