Tariff quota notices

This pattern is archived

When to use

Use this pattern to tell users about changes to existing Tariff quotas and new Tariff quotas.

How to design using this pattern

Do not make any changes to the body of the Tariff quota notice. It must be an exact copy of the text that’s been provided and must match EU legislation.

Document type

Detailed guide

Title

The title should be ‘[Description] (Tariff quota notice [number])’.

For example, ‘Agricultural and industrial products (Tariff Quota Notice 151)’.

Collection page

Add the URL of any new page to the Tariff quota notices collection. Do not create a public change note.
