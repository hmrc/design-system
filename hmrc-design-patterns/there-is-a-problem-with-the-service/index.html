Breaking change There has recently been a breaking change to the GOV.UK and HMRC design systems, please contact #team-plat-ui on Slack if you require more information or assistance in updating your prototype.

HMRC Design Patterns

  • Account header
  • Accounts office reference
  • Ask the user for their consent
  • Confirmed identity
  • Could not confirm identity
  • Employer PAYE reference
  • Notification badge
  • Open in a new window or tab
  • Page heading
  • Page not found
  • Page title
  • Service unavailable
  • There is a problem with the service
  • Unique Taxpayer Reference
  • VAT registration number
  • HMRC Design Patterns backlog
  • Installing HMRC Frontend in your prototype

There is a problem with the service

Use the There is a problem with the service pattern from the GOV.UK Design System.

The GOV.UK Design System does not contain Welsh translations yet. These examples include recommended content in Welsh.
