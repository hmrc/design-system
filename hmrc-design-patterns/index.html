HMRC Design Patterns

These are patterns designed specifically for the needs of HMRC users and patterns that are not yet in the GOV.UK Design System.

Check the GOV.UK Design System to see if the component or pattern you need is already being used across government.

If you cannot find what you need in the GOV.UK Design System or HMRC Design Patterns you can:

  • see if it is being worked on by other teams across government and add your findings
  • see if it is being worked on by HMRC and add your findings
  • propose a new pattern

All patterns must be useful and unique.

Using code snippets in prototypes

Each pattern includes coded examples. You will need to install HMRC Frontend to use patterns with specific HMRC code not yet supported by the GOV.UK Prototype Kit. You must use version 7 or later of the GOV.UK Prototype Kit for these to work.

Find out how to:

  • set up the Prototype Kit
  • update the Prototype Kit
  • update your code

Get help

HMRC Design Patterns is maintained by the Platform UI team. If you have a question, idea or suggestion you can:

  • email the Design System team
  • get in touch with #team-design-system on HMRC Slack (open in app)
